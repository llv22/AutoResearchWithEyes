# ICLR 2026 Template

Download the official ICLR 2026 author kit and place the style files here:
- `iclr2026_conference.sty`
- Any additional `.sty` or `.cls` files from the kit

## Download
Visit the ICLR 2026 submission page on OpenReview for the official author kit.

## Page Limit
- Main body: 9 pages (to end of Conclusion)
- References and appendix: unlimited

## Citation Style
Uses `natbib` (`\citep` / `\citet`)
