# EMNLP 2026 Template

Download the official EMNLP 2026 author kit and place the style files here:
- `acl.sty`
- `acl_natbib.bst`
- Any additional `.sty` or `.cls` files from the kit

## Download
Visit the EMNLP 2026 call for papers page or the ACL Rolling Review (ARR) for the official author kit.
EMNLP uses the standard ACL style files (typically from https://github.com/acl-org/acl-style-files).

## Page Limit
- Main body: 8 pages (long papers) / 4 pages (short papers), to end of Conclusion
- References and appendix: unlimited

## Citation Style
Uses `natbib` (`\citep` / `\citet`)
