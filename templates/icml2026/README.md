# ICML 2026 Template

Download the official ICML 2026 author kit and place the style files here:
- `icml2026.sty`
- Any additional `.sty` or `.cls` files from the kit

## Download
Visit https://icml.cc for the official author kit.

## Page Limit
- Main body: 8 pages (to end of Conclusion)
- References and appendix: unlimited

## Citation Style
Uses `natbib` (`\citep` / `\citet`)
